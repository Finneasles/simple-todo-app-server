## Prerequisites 
*    NodeJS
*    MongoDB

## Setup 
### Install dependencies 

``` bash
$ npm install
```

## Production & Development 
To start the server :
### Production
``` bash
$ npm start
```
To start the server with `nodemon` :
### Development
``` bash
$ npm run dev
```
